This animated navigation menu showcases creative hover interactions using just HTML and CSS. It's responsive, clean, and offers dynamic underline animations including center, upward reveal, and forward scale ‚Äî perfect for portfolios and modern interfaces.

Below is the HTML structure that sets up the menu layout. It consists of a flex-based unordered list with custom classes on list items to apply individual hover effects.

üìÑ HTML Code


  <ul class="nav-links">
    <li><a href="#">Dashboard</a></li>
    <li class="center"><a href="#">Portfolio</a></li>
    <li class="upward"><a href="#">Services</a></li>
    <li class="forward"><a href="#">Feedback</a></li>
  </ul>

The CSS utilizes pseudo-elements (`::before`) to create stylish line animations under each nav link. Each hover effect‚Äîwhether it grows from center, slides up, or expands from right to left‚Äîoffers a unique visual experience. Flexbox keeps layout aligned and responsive across devices.

üé® CSS Code


  .nav-links {
    display: flex;
    align-items: center;
    background: #fff;
    padding: 20px 15px;
    border-radius: 12px;
    box-shadow: 0 5px 10px rgba(0,0,0,0.2);
  }
  
  .nav-links li a {
    position: relative;
    color: #333;
    font-size: 20px;
    font-weight: 500;
    text-decoration: none;
  }
  
  .nav-links li a::before {
    content: '';
    position: absolute;
    bottom: 0;
    left: 0;
    height: 3px;
    width: 0%;
    background: #34efdf;
    border-radius: 12px;
    transition: all 0.4s ease;
  }
  
  .nav-links li a:hover::before {
    width: 100%;
  }
  
  /* Center hover */
  .nav-links li.center a::before {
    left: 50%;
    transform: translateX(-50%);
  }
  
  /* Upward reveal */
  .nav-links li.upward a::before {
    bottom: -5px;
    opacity: 0;
  }
  .nav-links li.upward a:hover::before {
    bottom: 0px;
    opacity: 1;
  }
  
  /* Forward scale */
  .nav-links li.forward a::before {
    transform: scaleX(0);
    transform-origin: right;
  }
  .nav-links li.forward a:hover::before {
    transform: scaleX(1);
    transform-origin: left;
  }
